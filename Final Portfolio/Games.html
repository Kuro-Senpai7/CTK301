Basic Mechanics

Compilation of some the basic mechanics that I learned to do in Unity both in 2D and 3D enviroments.

The order of the videos showcased are in this order: simple spawn and destory, simple sprite display and animation, projectiles, and enemies on a 2D environment, raycasting and affecting said raycast, using raycasting to pick up and drop items, turning on and off a mesh renderer using a key, and adding simple footsteps audio.

Treat Collection

A game with a theme about dogs featuring my dog as the background for the game, as well as music made by me, and it is about a dog who has an insatiable hunger for dog treats, and everytime he gets full, the dog gets more hungry the next time.

First 2D Project

First 2D project made in Unity. A game in space about surviving and getting the highest score.
