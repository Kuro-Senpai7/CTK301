First 2D Project

First 2D project made in Unity. A space invaders like game where there are random enemies spawning as well as asteroids, and the goal is to survive and score as high as possible. The character is on the bottom and can only move left and right, but it can shoot to gain score, and he has 3 lives before it is game over.
